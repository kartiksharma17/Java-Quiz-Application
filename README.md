# Java Quiz Application

Practicing a simple Quiz Application built using Java and Object-Oriented Programming concepts. The application presents multiple-choice questions, evaluates user answers, calculates the final score, and displays the result through a graphical user interface.

## Technologies Used

- Java
- Java Swing (GUI)
- OOP Concepts

## Concepts Practiced

- Classes and Objects
- Constructors
- Encapsulation
- Arrays
- Methods
- GUI Development with Swing

## Features

- Multiple-choice quiz questions
- Interactive GUI
- Answer validation
- Score calculation
- Final result display
- User-friendly interface

## Project Structure

```
src/
├── Question.java
├── QuizApplication.java
└── Main.java
└── QuizGUI.java
```
